# Fullstack Mastery - Module 2

Repository for module:
`database-integration-orm-odm-caching`

## Folder structure

- `0-sql-vs-nosql-in-nestjs`: orientation and comparison notes for SQL vs NoSQL in NestJS
- `1-typeorm-and-postgresql`: hands-on TypeORM + PostgreSQL project
- `2-mongoose-and-mongodb`: hands-on Mongoose + MongoDB project
- `3-caching-with-redis`: hands-on Redis caching project

## Quick start

```bash
cd 1-typeorm-and-postgresql
npm install
npm run start:dev
```

Use the same flow for `2-mongoose-and-mongodb` and `3-caching-with-redis`.
